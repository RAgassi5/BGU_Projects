## Tools and Libraries Used
The project uses the following tools and libraries (as listed in the `pom.xml` Maven configuration file):

- **org.slf4j:slf4j-api** - logfile from lab
- **org.xerial:sqlite-jdbc** – used for SQLite database access
- **org.junit.jupiter:junit-jupiter** – JUnit testing framework
- **org.assertj:assertj-core:3.24.2** – fluent assertions for testing
- **org.mockito:mockito-core:5.5.0** – mocking framework for unit tests

## Database
The project uses an SQLite database (`ADSS_Group_AR.db`).
